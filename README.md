# Bancos

## Ejecutar local

1. Crear/activar entorno virtual.
2. Instalar dependencias:

```bash
pip install -r requirements.txt
```

3. Ejecutar app:

```bash
python main.py
```

### Ejecutar en Windows (doble clic)

También puede ejecutar `iniciar_app.bat` para abrir la aplicación sin usar comandos.
